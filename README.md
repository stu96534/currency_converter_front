# 貨幣轉換器
以爬蟲技術取得台灣銀行貨幣匯率資料，用前後端分離方式打造，此為前端專案。
- 部屬網站為vercel 網址: 
- 後端專案連結：

## 功能介紹
### 貨幣查詢
 - 可即時查詢台灣銀行貨幣匯率資料
### 貨幣轉換
 - 可輸入台幣金額換算外幣金額
 - 可輸入外幣金額換算台幣金額
 - 可選擇使用現金買入、現金賣出、即期買入、即期賣出之匯率進行換算

## 安裝

```sh
npm install
```
